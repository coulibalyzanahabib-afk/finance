## 💻 Installation & Utilisation

### Option 1 — En ligne (GitHub Pages)
1. Fork ce dépôt
2. Aller dans **Settings → Pages**
3. Source : `Deploy from a branch` → `main` → `/ (root)`
4. Votre app sera disponible sur `https://VOTRE_USERNAME.github.io/hcfina`

### Option 2 — En local
1. Télécharger `index.html`
2. Double-cliquer pour ouvrir dans Chrome / Edge / Firefox
3. Aucune installation requise ✅
